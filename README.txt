Alunos:

Sofia Barata

Projeto:

Criação de um website com tema de studenthub que tinha como objetivo seguimento de um "main" website com 4 páginas: Inicio / eventos /serviços / contactos.
Com adição de login page e páginas para user e administrador.

O codigo presente no projeto apresenta ser interativo e dinânico, estando o mesmo comentado para melhor entendimento do mesmo.

Extras:
Criação de página html para erros 404 (de modo a não apresentar uma página default)
Reedirect automatizado para a página personalizada de 404 em caso do user ir para uma página inexsitente
Deploy do website num servidor com uso de servidor de DNS (iplusons.ddns.net) -> Iplusons (Ipluso Nuno Sofia).
